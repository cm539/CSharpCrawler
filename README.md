# CSharpCrawler

Простой веб-краулер на C#. Обходит сайты из списка, собирает статистику: количество ключевых слов на каждой странице.  
Текущая версия обходит только те сайты, у которых есть robots.txt и sitemap.  
Все данные сохраняются в БД (MS SQL Server 2014 или MySQL Server 5.7 на выбор), применяется подход Code First.

### Быстрый старт
1. Открыть solution в Visual Studio 2015
2. Поменять connectionString
3. Запустить приложение  

или  

1. Скачать бинарники (папка /bin)
2. Поменять connectionString
3. Запустить приложение

Для примера производится обход сайта lenta.ru на наличие ключевого слова "путин".

### Используемые библиотеки
- EntityFramework 6  
- MySQL Connector Net 6.9.99  
- Ninject 3.2
- AngleSharp 0.9.9
